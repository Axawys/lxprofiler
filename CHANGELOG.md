Этот файл показывается при уведомлении об обновлении — перечисляется всё новое
из версий между установленной у пользователя и доступной в репозитории.

## 4.13.0
- Учёт реального использования инструментов: CLI вроде nmap, sqlmap, docker,
  terraform, gpg, openvpn и т.п. засчитываются только при следе в истории команд;
  nginx — при следе в истории, активном/включённом сервисе или логах доступа.
- Инструменты без следа использования (GUI вроде Wireshark/Burp, демоны) и случаи
  без доступной истории по-прежнему учитываются по наличию — ничего не ломается.

## 4.12.0
- Новый скрытый класс «Любитель легаси» (показывается при заполненности >20%):
  старое ядро Linux, 32-битная система, X11/Xorg, древние DE/WM (Trinity, MATE,
  FVWM, IceWM, Window Maker и т.п.), легаси-софт (MPlayer, Pidgin, XMMS).

## 4.11.0
- Команда `lxprofile -c` / `--changes [версия]`: показывает changelog указанной
  версии, а без аргумента — пяти последних версий.

## 4.10.1
- Java теперь учитывается в профиле программиста при следах разработки
  (JDK `javac`, Maven/Gradle, `~/.m2`/`~/.gradle`, jenv/SDKMAN), а не только
  как рантайм.

## 4.10.0
- Учёт каталога `Games` (без учёта регистра) с вручную установленными играми
  (TLauncher/Minecraft, War Thunder и т.п.) — очки геймеру.
- В уведомлении об обновлении показывается changelog: всё новое из всех версий
  между вашей и доступной.

## 4.9.2
- Управление работает и при включённой русской раскладке.

## 4.9.1

## 4.8.0

  без изменений.

## 4.7.3
